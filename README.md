# ✦ Closet Couture

A beautiful AI-powered personal wardrobe & outfit builder app.

## Features
- **Digital Wardrobe** — Upload clothes organised by category (Tops, Bottoms, Dresses, Outerwear, Shoes, Accessories, Bags)
- **Visual Outfit Builder** — Pick items from each category; a real outfit image is generated and saved
- **Couture Buddy AI** — Describe your vibe/inspo and the AI picks from YOUR wardrobe to create an outfit image
- **Gender-personalised UI** — Barbie × Popstar pink/purple theme for feminine, Old Money blue-grey for masculine
- **Outfit Gallery** — Browse, download, and delete all your outfits

## Setup

```bash
pip install -r requirements.txt
python app.py
```

Then open http://localhost:5000

## Notes
- Outfit images are generated by compositing your uploaded clothing photos using Pillow
- Couture Buddy calls the Anthropic Claude API — the API key is handled by the environment (no key needed in code)
- All data stored in SQLite (`closet.db`) and uploads in `static/uploads/`

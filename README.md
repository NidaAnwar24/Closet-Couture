✦ Closet Couture

A personal AI style studio built with Python & Flask.

Upload your clothes, organise them by category, and build outfits visually 
by mixing and matching your actual wardrobe pieces. Let Couture Buddy — the 
built-in AI stylist — create a complete outfit image from your closet based 
on a vibe or inspiration you describe.

Features a gender-personalised UI: a Barbie × Popstar pink/purple theme 
for feminine users and an Old Money blue-grey aesthetic for masculine users.

Built with Flask, SQLAlchemy, Pillow, and the Claude API.
